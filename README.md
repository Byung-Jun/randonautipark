
# Randonautipark
```
Randonautica와 비슷하게 구현해보고자 했습니다
만, 랜덤하게 지도 api를 찍게만 해두었습니다.
```

## 계획 
원래 계획은 채팅기능을 추가하고,
어둡고 기괴한 분위기의 색상을 선택하여,
랜덤하게 선택한 지역의 로드뷰를 가져와
그 색과 해당 색상과 유사도를 파악하여
일정 유사도가 넘으면 나타내려고 했습니다.
추후에 계속적으로 업데이트 해보겠습니다.
